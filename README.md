# 📊 Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow, starting from raw data and ending with interactive dashboards and business insights. The project includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, Power BI visualization, and final reporting.

The objective is to transform raw data into meaningful insights that support better decision-making using modern data analytics tools.

---

## Dataset

The dataset contains structured business data used for analysis. It includes multiple attributes such as customer information, transactions, products, sales, and other relevant business metrics.

**Key Features**

* Customer details
* Product information
* Sales records
* Transaction data
* Business performance metrics

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Inspected the data structure and data types.
* Identified missing values and duplicates.

### 2. Exploratory Data Analysis (EDA)

* Performed descriptive statistics.
* Analyzed distributions and trends.
* Created visualizations to understand the data.
* Identified relationships between variables.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data.
* Converted data into appropriate formats.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into a relational database.
* Executed SQL queries using PostgreSQL, MySQL, and SQL Server.
* Used:

  * SELECT
  * WHERE
  * GROUP BY
  * ORDER BY
  * JOIN
  * Aggregate Functions
  * Subqueries
* Extracted business insights from the database.

### 5. Power BI Dashboard

Developed an interactive dashboard featuring:

* KPI Cards
* Sales Analysis
* Customer Insights
* Product Performance
* Trend Analysis
* Interactive Filters and Slicers

### 6. Report Creation

Prepared a comprehensive analytics report summarizing:

* Data preparation process
* Analysis methodology
* Key findings
* Business recommendations

### 7. Presentation

Created a professional presentation using **Gamma** to communicate the project objectives, methodology, dashboard, findings, and recommendations.

## How to Run

1. Clone this repository.
2. Open the Jupyter Notebook.
3. Install the required Python libraries.
4. Load the dataset.
5. Run the notebook cells sequentially.
6. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
7. Execute the SQL scripts.
8. Open the Power BI (.pbix) file to explore the dashboard.
9. Review the report and presentation for project insights.
